# StudyNook

### Live Site URL
https://stufynook-project-9.vercel.app

---

## Description
StudyNook is a modern study space booking platform that helps students easily find, book, and manage study rooms. Users can search rooms by name, price, and amenities, book instantly with real-time availability, and even list their own study spaces to earn income.

---

## Features

### 1. Browse & Search Study Rooms
- Users can search and filter study rooms by name, price, and amenities.
- Helps users quickly find the most suitable study environment.
- Improves discovery with a simple and intuitive interface.

### 2. Instant Booking System
- Users can book study rooms with real-time availability checking.
- Prevents double booking and ensures smooth reservation experience.
- Saves time with fast and reliable booking flow.

### 3. User Dashboard
- Users can manage bookings in one place.
- Helps track active, upcoming, and past reservations easily.
- Provides a centralized control panel for all activities.

### 4. Host Your Study Space
- Users can list their own study rooms on the platform.
- Enables earning income by renting study spaces hourly.
- Makes the platform community-driven.

### 5. Fully Responsive Design
- Works smoothly on mobile, tablet, and desktop devices.
- Ensures consistent experience across all screen sizes.
